# Running the Cricbuzz Livematch Scraper

## Execute: 
`node main.mjs`

## Result: 
A JSON data structure will be printed to the screen in the following format:
```json
{
  "GMT-date": [
    {
      "match-name": "string",
      "match-url": "string",
      "match-GMT-time": "string"
    }, // ...
  ], // ...
}
```
